# simple-webscraping

## Description

This is a simple webscraping project using Python and BeautifulSoup4.

## How to use

1. Clone this repository

   ```bash
   git clone 
   ```

2. Install the requirements

   ```bash
   pip install -r requirements.txt
   ```

3. Run the script

   ```bash
   python app.py
   ```
